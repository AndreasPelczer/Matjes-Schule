# Matjes-Schule

**"Matjes, der kleine Hering" - Schulversion**

Das Ausbildungsspiel der Kueche - Profi-Edition fuer Berufsschulen und Ausbilder.

## Was ist das?

Eine iOS-App fuer Koch/Koechin-Ausbilder an Berufsschulen. Schueler spielen das Matjes-Quiz und Ausbilder verfolgen den Lernfortschritt.

### Features

**Fuer Schueler:**
- 20 Level in 4 Halbjahren mit ~566 Fragen
- Lexikon: 134 Produkte, 20 Garmethoden, 20 Saucen
- Sterne-System (0-3 pro Level)
- Commis-Pruefung + Bossfight
- Bloom-Taxonomie (Erkennen -> Zuordnen -> Wissen -> Anwenden -> Bewerten)

**Fuer Ausbilder:**
- Klassen und Schueler verwalten
- Lernfortschritte aller Schueler einsehen
- Eigene Fragenkataloge erstellen
- Fortschrittsberichte exportieren (PDF)
- Biometrie + PIN Login

### Technologie

- Swift / SwiftUI
- MVVM-Architektur
- iOS 17.0+
- CloudKit (fuer Sync)
- Keine externen Abhaengigkeiten
